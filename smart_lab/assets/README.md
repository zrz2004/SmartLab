# Assets Directory

This directory contains static assets for the SmartLab application.

## Structure

```
assets/
├── images/     # PNG, JPG images
├── icons/      # SVG icons
└── fonts/      # Custom fonts (PingFang, DIN Alternate)
```

## Required Fonts

Download and place the following fonts in `assets/fonts/`:

1. **PingFang SC** (Chinese UI text)
   - PingFang-Regular.ttf
   - PingFang-Medium.ttf
   - PingFang-Bold.ttf

2. **DIN Alternate** (Numeric displays)
   - DINAlternate-Bold.ttf

## Notes

- Use SVG format for icons when possible
- Optimize images before adding (use tools like TinyPNG)
- Follow 1x, 2x, 3x naming convention for raster images
